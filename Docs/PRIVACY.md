# Privacy Policy

Last updated: 2026-07-01

VirtConnector is a macOS menu bar utility that mirrors Mac power events to a user-configured Matter-capable device.

## Data Collection

VirtConnector does not collect, transmit, sell, or share personal data.

## Local Data

VirtConnector stores configuration locally on the Mac, including login-at-startup preference, selected device identifiers, and the last requested power state. This data remains on the device unless the user exports or backs it up through macOS features.

## Smart Home Access

VirtConnector may request smart home access in builds that include a HomeKit-capable bridge. Access is used only to discover or control user-selected accessories and is controlled by macOS and Apple account permissions.

## Network Use

VirtConnector may communicate with smart home devices on the local network to send On/Off requests. It does not use analytics, advertising SDKs, or remote logging services.

## Contact

For privacy questions, contact the maintainer through the project repository or App Store support channel.

# プライバシーポリシー

最終更新日: 2026-07-01

VirtConnectorは、Macの電源イベントをユーザーが設定したMatter対応デバイスへ反映するmacOSメニューバーユーティリティです。

## データ収集

VirtConnectorは個人データを収集、送信、販売、共有しません。

## ローカルデータ

VirtConnectorは、ログイン時起動の設定、選択されたデバイス識別子、最後に要求した電源状態などの設定をMac上に保存します。このデータは、ユーザーがmacOSの機能で書き出しまたはバックアップしない限り、端末内に留まります。

## スマートホームアクセス

VirtConnectorは、HomeKit対応bridgeを含むビルドでスマートホームアクセスを要求する場合があります。アクセスは、ユーザーが選択したアクセサリを検出または制御する目的でのみ使用し、macOSおよびAppleアカウントの権限管理に従います。

## ネットワーク利用

VirtConnectorは、On/Off要求を送るためにローカルネットワーク上のスマートホームデバイスと通信する場合があります。分析、広告SDK、リモートログサービスは使用しません。

## 連絡先

プライバシーに関する問い合わせは、プロジェクトリポジトリまたはApp Storeのサポート窓口から行ってください。
