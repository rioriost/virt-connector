# VirtConnector

VirtConnector is a macOS menu bar utility that mirrors Mac power events to a user-configured Matter-capable device.

## Goals

- Wake maps to device On.
- Sleep and Power Off map to device Off.
- The app runs as a menu bar app.
- Login-at-startup is controlled by an explicit user setting.
- UI strings are localized in English and Japanese.
- The app is prepared for App Store distribution with sandbox and HomeKit entitlements.

## Current Implementation

This repository now contains a native macOS Xcode project generated by XcodeGen:

- `NSWorkspace.didWakeNotification`
- `NSWorkspace.willSleepNotification`
- `NSWorkspace.willPowerOffNotification`
- `NSWorkspace.extendPowerOff(by:)` for a short best-effort Power Off delay
- `SMAppService.mainApp` for the user-controlled login item setting

The Matter control surface is isolated behind `DevicePowerControlling`. The current implementation records requests and returns an unsupported-backend error until Matter pairing/access is wired to the selected device.

## Build

```sh
xcodegen generate
xcodebuild -project VirtConnector.xcodeproj -scheme VirtConnector -configuration Debug CODE_SIGNING_ALLOWED=NO build
```

For App Store distribution, configure the Apple Developer App ID `st.rio.virt-connector` with the required capabilities, select a valid development team in Xcode, and archive from Xcode.
